# CMake_GTest

A demo project using CMake to build and GTest to do unit test.

## What we USE in this project

1. CMake
2. GoogleTest

## Dependencies

### 1.CMake

```shell
#install 
sudo apt install cmake

#unistall 
sudo apt autoremove cmake 
```

### 2.GoogleTest

```shell
sudo apt install libgtest-dev
```
